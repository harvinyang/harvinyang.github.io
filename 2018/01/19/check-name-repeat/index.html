check_name_repeat

注册检测用户名是否相同，提示用户名重复，使用focusout检测失去焦点触发，ajax读出name在数据库中是否存在，如果存在namerepeat = 1，否则 namerepeat = 0；
在点击提交时检测namerepeat = 1，提示用户重新检测重复内容，namerepeat = 0 ，允许提交

当修改name是再次检测name是否冲突。重复注册检测步骤
