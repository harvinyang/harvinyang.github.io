React區分使用state和props

props是一種從父級向子級傳遞數據的方法，state只在交互的時候使用

1 它是通過props從父級傳來的嗎？如果是，它可能不是state。
2 它隨着時間推移不變嗎？如果是，它可能不是state
3 你能過根據組件中任何其他的state或props把它計算出來嗎？如果是，它不是state
